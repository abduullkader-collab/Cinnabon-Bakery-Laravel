# Cinnabon Bakery E-Commerce

## Project Overview
This is a full-stack e-commerce website built for Cinnabon Bakery, demonstrating advanced web development skills in building scalable, secure, and user-friendly online stores. The project showcases the ability to create a complete digital marketplace from scratch, handling everything from product display to payment processing.

## What the Project Consists Of
- **Product Catalog**: Dynamic product pages with category filtering, image galleries, and detailed pricing information.
- **Shopping Cart**: Session-based cart management allowing users to add, update, and remove items with quantity controls.
- **Secure Checkout**: Integrated Stripe payment gateway for safe and reliable transaction processing.
- **Order Management**: Complete order tracking system with database storage and success confirmations.
- **Admin Panel**: Backend interface for managing products, categories, and orders (authentication required).
- **User Interface**: Responsive design optimized for desktop, tablet, and mobile devices.
- **Additional Pages**: About, Contact, and News sections for enhanced user engagement.

## Skills Demonstrated
- **Full-Stack Development**: Proficient in both frontend and backend technologies, creating seamless user experiences.
- **Laravel Framework Expertise**: Advanced use of Laravel 9 for MVC architecture, routing, middleware, and database interactions.
- **Database Design**: Relational database modeling with MySQL, including migrations, seeders, and Eloquent ORM.
- **Payment Integration**: Secure implementation of Stripe API for e-commerce transactions.
- **Frontend Technologies**: Responsive web design with Bootstrap, jQuery for interactivity, and custom CSS.
- **Security Best Practices**: CSRF protection, input validation, and secure session management.
- **Version Control**: Git-based development with proper commit history and project structure.
- **Problem-Solving**: Ability to architect complex systems, handle edge cases, and optimize performance.

## Tech Stack
- **Backend**: Laravel 9 (PHP 8), MySQL
- **Frontend**: Bootstrap, jQuery, Custom CSS
- **Payments**: Stripe API
- **Development Tools**: Composer, Artisan, PHPUnit for testing
